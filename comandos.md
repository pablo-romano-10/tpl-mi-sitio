# comandos

**El comando pwp**( "Print Working Directory" (Imprimir Directorio de Trabajo)

**El comando ls**(abreviatura de List) sirve para listar el contenido de la carpeta en la que estás parado

**El comando cd** (Change Directory) es el que usás para moverte entre carpetas desde la termina

**El comando dkdir** (crea una carpeta)

**El comando touch** se usa principalmente para crear archivos vacíos de forma instantánea desde la terminal

**El comando cat** (abreviatura de concatenate) se usa para leer y mostrar el contenido de un archivo directamente en la terminal, sin necesidad de abrir un editor como VS Code.

**El comando clear**( para limpiar el terminal )
